QuizForge REST endpoints

Use these JSON endpoints to power mobile apps, dashboards, or AJAX interactions. Public reads are open; content mutation requires staff access.

Authentication

Session-based authentication with CSRF support for browser clients and JSON responses for frontend apps.

  • Register and login APIs
  • OTP password reset flow
  • Profile and dashboard endpoints

Quiz Resources

REST-style quiz and question endpoints include different HTTP methods for clean CRUD behavior.

  • GET for list/detail
  • POST for create/submit
  • PUT, PATCH, DELETE for staff edits

Results

Submit answers, calculate scores, read results, and build leaderboard experiences from the API.

  • Score summaries
  • Leaderboard data
  • Dashboard analytics
